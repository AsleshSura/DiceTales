## üîÑ Data Flow Patterns

### Application Initialization Flow
```
1. index.html loads ‚Üí Script loading cascade
2. config.js ‚Üí AI_CONFIG global
3. utils.js ‚Üí Global utility functions
4. gameState.js ‚Üí GameState instance creation
5. character.js ‚Üí CharacterManager instance
6. dice.js ‚Üí DiceSystem instance
7. dmEvaluator.js ‚Üí DMEvaluator class definition
8. ai.js ‚Üí AIManager with DMEvaluator integration
9. audio.js ‚Üí AudioManager instance
10. ui.js ‚Üí UIManager instance
11. main.js ‚Üí DiceTalesApp initialization
```

### Character Creation Data Flow
```
User Input ‚Üí CharacterManager ‚Üí Validation ‚Üí GameState ‚Üí LocalStorage
    ‚Üì              ‚Üì               ‚Üì            ‚Üì            ‚Üì
  Form Data ‚Üí Step Processing ‚Üí Data Validation ‚Üí State Update ‚Üí Persistence
    ‚Üì              ‚Üì               ‚Üì            ‚Üì            ‚Üì
Character ‚Üí Setting + Class + ‚Üí Complete ‚Üí gameState.set() ‚Üí Auto-save
  Data        Stats + Details    Character    Character      Triggered
```

### Game Loop Data Flow
```
Player Action ‚Üí UI Processing ‚Üí AI Generation ‚Üí Response Processing ‚Üí Display Update
     ‚Üì              ‚Üì               ‚Üì                ‚Üì                ‚Üì
Text/Dice ‚Üí uiManager.submit ‚Üí aiManager.process ‚Üí dmEvaluator.score ‚Üí UI Refresh
  Input       PlayerAction      PlayerAction       Response           + State Save
     ‚Üì              ‚Üì               ‚Üì                ‚Üì                ‚Üì
  Validation ‚Üí Action Parsing ‚Üí HuggingFace ‚Üí Quality Assessment ‚Üí Screen Update
     ‚Üì              ‚Üì              Request           ‚Üì                ‚Üì
  Required ‚Üí Context Building ‚Üí API Response ‚Üí Feedback Generation ‚Üí Next Turn
```

### State Persistence Flow  
```
Any State Change ‚Üí gameState.set() ‚Üí Internal Update ‚Üí Auto-save Trigger ‚Üí localStorage
       ‚Üì                ‚Üì                 ‚Üì               ‚Üì                 ‚Üì
   User Action ‚Üí Path-based Update ‚Üí Validation ‚Üí Debounced Save ‚Üí JSON Serialization
       ‚Üì                ‚Üì                 ‚Üì               ‚Üì                 ‚Üì
  UI Interaction ‚Üí Nested Property ‚Üí Type Checking ‚Üí 5s Delay ‚Üí Browser Storage
       ‚Üì                ‚Üì                 ‚Üì               ‚Üì                 ‚Üì
Setting Change ‚Üí Deep Object Merge ‚Üí Range Validation ‚Üí Batch Save ‚Üí Persistence
```

### GameState API
```javascript
// Core State Management
gameState.get(path)                      // Get value by path
gameState.set(path, value)               // Set value by path  
gameState.update(path, partialObject)    // Merge update
gameState.push(path, item)               // Add to array
gameState.remove(path, predicate)        // Remove from array

// Character Management
gameState.getCharacter()                 // Get character object
gameState.setCharacter(character)        // Set entire character
gameState.updateCharacterStat(stat, value) // Update single stat

// Campaign Management  
gameState.getCampaign()                  // Get campaign object
gameState.setCampaign(campaign)          // Set entire campaign
gameState.addToCampaignLog(entry)        // Add log entry

// Settings Management
gameState.getSetting(path)               // Get setting value
gameState.setSetting(path, value)        // Set setting value

// Persistence
gameState.save()                         // Manual save
gameState.load()                         // Load from storage
gameState.reset()                        // Reset to defaults
gameState.export()                       // Export as JSON
gameState.import(jsonString)             // Import from JSON

// Validation
gameState.isCharacterCreated()           // Check character exists
gameState.isCampaignStarted()           // Check campaign active
gameState.canStartGame()                // Check ready to play

// Utility
gameState.getStateSize()                // Get storage size
gameState.getStateInfo()                // Get metadata
```

### LocalStorage API Usage
```javascript
// Storage Keys
const STORAGE_KEYS = {
    GAME_STATE: 'dicetales_gamestate',
    SETTINGS: 'dicetales_settings',
    CHARACTER: 'dicetales_character',
    CAMPAIGN: 'dicetales_campaign'
};

// Storage Operations
localStorage.setItem(key, JSON.stringify(data));
const data = JSON.parse(localStorage.getItem(key) || '{}');
localStorage.removeItem(key);
localStorage.clear();

// Storage Event Handling
window.addEventListener('storage', (event) => {
    if (event.key === STORAGE_KEYS.GAME_STATE) {
        // Handle external state changes
        gameState.load();
    }
});
```
